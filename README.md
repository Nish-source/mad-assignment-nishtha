# MAD Assignment

This repository contains my Mobile Application Development (MAD) assignment. It includes 4 Android applications covering different concepts like UI design, media handling, sensors and storage.

---

## 📱 Projects

### 1. Currency Converter App

This application is used to convert currencies between INR, USD, EUR and JPY.

Features:

* User can enter amount and select currencies
* Conversion between 4 currencies (INR, USD, EUR, JPY)
* Simple and clean user interface
* Settings option available to change theme

  * Light Mode
  * Dark Mode

Concepts used:

* UI design using XML
* Activity navigation
* Theme switching

---

### 2. Media Player App

This app is used to play audio files from the device and stream video from the internet.

Features:

* Play audio file from storage
* Stream video using URL
* Media control buttons:

  * Open File
  * Open URL
  * Play
  * Pause
  * Stop
  * Restart

Sample video URL used:
https://media.w3.org/2010/05/sintel/trailer.mp4

Concepts used:

* MediaPlayer / VideoView
* Handling media controls
* Working with online URLs

---

### 3. Sensors App

This application reads and displays data from mobile sensors in real-time.

Sensors implemented:

* Accelerometer
* Light Sensor
* Proximity Sensor

Features:

* Displays live sensor values
* Updates automatically when values change
* Simple UI for monitoring

Concepts used:

* SensorManager
* Real-time data handling
* Event listeners

---

### 4. Gallery & Camera App

This is a basic gallery application with camera functionality.

Features:

📸 Camera:

* Capture images using device camera
* Save images to selected folder
* Handles runtime permissions

🖼️ Gallery:

* Select a folder from storage
* Display images in grid format

📄 Image Details:

* Shows image name, path, size and date
* Opens when image is clicked

🗑️ Delete Option:

* Delete image using button
* Confirmation dialog before deleting
* Returns to gallery after deletion

Concepts used:

* Camera API
* File handling and storage
* RecyclerView/Grid layout
* Permissions handling
